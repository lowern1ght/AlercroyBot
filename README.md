# AlercroyBot

> Useful telegram bot

### Getting started

The **first** thing to do is to set the token in the configuration file

![token config](Repository/scr_1.png)

**or** command line arguments

```c#
 dotnet run Token="U_TOKEN"
```

Open **command list**

![command list](Repository/scr_2.png)
